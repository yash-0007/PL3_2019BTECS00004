Program for Celsius To Fahrenheit conversion

Formula for converting Celsius scale to Fahrenheit scale

T(oF) = T(oC) Ã— 9/5 + 32

Code

#include 
int main()
{
    float fahrenheit, celsius;
    printf("Enter temperature in celsius: ");
    scanf("%f", &celsius);
    fahrenheit = ((celsius * 9) / 5) + 32;
    printf("Temperature in fahrenheit is:  %f", fahrenheit);
    return 0;
}

Sample input

Enter temperature in celsius: 24

Sample output

Temperature in Fahrenheit is: 75.1999997

Variable used in the program

fahrenheit celsius
